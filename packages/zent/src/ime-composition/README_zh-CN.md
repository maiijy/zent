## IMEComposition

`IMEComposition` 的子 `Input` 组件在输入法输入时的 `onChange` 事件，使其不会在每次输入时都被触发，而是在输入完成后才被触发。

### API

| 参数   | 说明 | 类型 | 是否必须 | 默认值 | 备选值 |
| ------ | ---- | ---- | -------- | ------ | ------ |
| enable |      | bool | 否       | false  |        |
